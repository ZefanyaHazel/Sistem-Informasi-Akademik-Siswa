# 📘 Sistem Informasi Akademik Siswa

Sistem Informasi Akademik Siswa (SIAKAD) adalah aplikasi berbasis web yang dirancang untuk membantu warga sekolah dalam mencari dan mengelola data siswa, guru, jadwal pelajaran, nilai rapor, dan berbagai informasi akademik lainnya secara efisien, terpusat, dan terstruktur.

## 🚀 Fitur Utama

- 🔐 Login multi-level (Admin, Guru, Siswa)
- 🎓 Manajemen data siswa dan guru
- 🗓️ Penjadwalan mata pelajaran
- 📊 Rapor bayangan dan nilai siswa
- 📁 Manajemen profil dan pengaturan pengguna
- 📈 Dashboard statistik akademik

## 🛠️ Teknologi yang Digunakan

- HTML, CSS (Native), JavaScript (Native)
